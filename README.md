# 🐍 Python Basics — Data Technician Bootcamp Project

This repository contains my Python fundamentals project completed during Week 6 of the **Data Technician Bootcamp**. The exercises were based on guided Google Colab notebooks and hands‑on tasks, including classic programming challenges such as **FizzBuzz**.  
The project demonstrates my ability to write clean, logical Python code while interacting with users, performing calculations, and controlling program flow.

---

## 🚀 What I Learned

### 🔤 Variables & Data Types  
I practised creating and manipulating variables to store text, numbers, and user‑provided values. This included understanding how Python handles different data types and how to convert between them when needed.

### 🖨️ Input & Output (`print()` and `input()`)  
Throughout the exercises, I built small programs that interact with users—collecting input, processing it, and displaying meaningful results.  
Examples included simple calculators, message prompts, and decision‑based outputs.

### 🔄 Type Casting  
To ensure correct calculations and comparisons, I used type casting functions such as:  
- `int()`  
- `float()`  
- `str()`  

This allowed me to convert user input into the appropriate format for mathematical operations or string manipulation.

### 🔀 Conditional Logic (`if`, `elif`, `else`)  
I used conditional statements to build decision‑making logic within programs.  
This included tasks such as:  
- Checking numeric conditions  
- Validating user input  
- Implementing the full logic for the **FizzBuzz** challenge  
  > “If a number is divisible by 3, print 'fizz'. If divisible by 5, print 'buzz'. If divisible by both, print 'fizzbuzz'.” 

### 🔁 Loops (`for` and `while`)  
I practised using loops to repeat actions efficiently, including:  
- Iterating through ranges of numbers  
- Repeating prompts until valid input is provided  
- Looping from 1 to 100 to generate the FizzBuzz output  
  > “Go through the integers from 1 to 100… Use loops and if statements.”
